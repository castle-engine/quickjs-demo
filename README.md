# quickjs-demo
Demo of QuickJS-Pascal in Castle Game Engine

This demo uses QJS to export some native pascal functions from the game to JS so you can use JS to interact with the game.

- Tested 
    - Mac OS Catalina
    - IOS 13.3.1
